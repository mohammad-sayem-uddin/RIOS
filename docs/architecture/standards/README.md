# Architecture Standards

Editorial and documentation standards governing the RIOS architecture
repository.

## Documents

| File                                           | Description                                           |
| ---------------------------------------------- | ----------------------------------------------------- |
| [Editorial-Standard.md](Editorial-Standard.md) | Editorial standard for all architecture documentation |

## Purpose

Standards ensure:

- Consistent document structure and formatting
- Uniform terminology across all architecture documents
- Clear authoring guidelines for contributors
- Machine-readable document structure for AI agents

## Usage

All architecture documentation must comply with the Editorial Standard.
Reference this document when creating or modifying any architecture artifact.
